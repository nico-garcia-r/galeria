# Galería de Arte · Cristina Daniele

Galería 3D interactiva en homenaje a mi abuela, la artista Cristina Daniele. Permite recorrer un pasillo virtual con 12 de sus obras, ver los detalles de cada cuadro y disfrutar de su trabajo desde cualquier dispositivo.

## Cómo usar

**Desktop:** flechas o WASD para moverse, mouse para mirar, clic en una obra para ver su información.

**Mobile:** botones en pantalla para moverse, swipe para mirar, toque en una obra para ver su información.

## Tecnologías

- [Three.js](https://threejs.org/) — renderizado 3D
- HTML, CSS y JavaScript puro — sin frameworks ni build step

## Correr localmente

Cualquier servidor estático sirve. Por ejemplo con Node.js:

```bash
npx serve .
```

Luego abrir `http://localhost:3000`.
